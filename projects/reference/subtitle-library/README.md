# 字幕ライブラリ

全 Udemy 講座（制作中・公開済み）の字幕ファイルを一元管理しています。

## 用途

### 新規講座制作時
- 過去の完成講座から構成・スタイルを参考にする
- 字幕の表現方法・説明スタイルを学ぶ
- ベストプラクティスのテンプレート化

### 講座更新判定時
- 公開中の講座の字幕から、更新が必要かどうかを判定
- 情報の古さ・技術トレンドとのズレを分析
- メンテナンス優先度を決定

### その他
- 字幕ファイルの再利用・改編
- 講座制作のベストプラクティス蓄積

## フォルダ構成

各講座フォルダには以下が含まれます：

```
[講座名]/
├── ja.vtt              # 日本語字幕
├── en.vtt              # 英語字幕（該当時）
├── es.vtt              # スペイン語字幕（該当時）
└── lecture-outline.md  # 講座概要・レクチャー一覧
```

## 参考資料

- 字幕フォーマット: WebVTT (.vtt)
- 字幕ガイド: [guidelines/course-creation.md](../../guidelines/course-creation.md) - 動画品質基準セクション
- 講座開発プロセス: [agents/udemy-course/course-developer.md](../../agents/udemy-course/course-developer.md)

## 使い方

新規講座作成時：
1. 参考にしたい講座フォルダを開く
2. `lecture-outline.md` で構成を確認
3. 字幕ファイルを参考にスタイルを統一
