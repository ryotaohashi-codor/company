# 字幕ライブラリ

過去に制作した Udemy 講座の字幕を参照用に保存しています。

## 用途

- 新規講座制作時の参考（表現・説明スタイル、構成パターン）
- 字幕ファイルの再利用・改編
- 講座制作のベストプラクティス学習

## フォルダ構成

各講座フォルダには以下が含まれます：

```
[講座名]/
├── ja.vtt              # 日本語字幕
├── en.vtt              # 英語字幕（該当時）
├── es.vtt              # スペイン語字幕（該当時）
└── lecture-outline.md  # 講座概要・レクチャー一覧
```

## 講座一覧

| 講座名 | リリース | 対象 | 字幕言語 |
|--------|---------|------|---------|
| [講座例] | 2026-01 | [学習者] | ja, en |

## 参考資料

- 字幕フォーマット: WebVTT (.vtt)
- 字幕ガイド: [guidelines/course-creation.md](../../guidelines/course-creation.md) - 動画品質基準セクション
- 講座開発プロセス: [agents/udemy-course/course-developer.md](../../agents/udemy-course/course-developer.md)

## 使い方

新規講座作成時：
1. 参考にしたい講座フォルダを開く
2. `lecture-outline.md` で構成を確認
3. 字幕ファイルを参考にスタイルを統一
